    Coupons

    During order synchronization, the WooCommerce Connector applies coupons from webstore orders to the respective sales orders in Business Central. You can view coupon details on the Coupon page.

    To view coupon information for a sales order

    1. Choose , enter sales orders, and then choose the related link.

      The Sales Orders page opens.

    2. Select the sales order for which you want to view coupon information, and then on the action bar, choose Manage > View.

      The Sales Order page opens.

    3. On the action bar, choose Order > Coupons.

      The Coupon page opens.

    Coupon FastTab

    The following list includes information about the fields on the Coupons FastTab:

    • Coupon Code: Specifies the name of the coupon.
    • Description: Specifies the description of the coupon.
    • Discount Type: Specifies the type of discount, for example, Percentage, Fixed Cart, Fixed Product.
    • Amount: Specifies the coupon amount.
    • Usage Count: Specifies the number of times coupons have been used.
    • Free Shipping: Specifies whether the coupon grants free shipping.
    • Expiry Date: Specifies the expiration date of the coupon.

    Usage Restriction FastTab

    The following list includes information about the fields on the Usage Restriction FastTab:

    • Minimum Spend: Specifies the minimum amount that must be spent to use the coupon.
    • Maximum Spend: Specifies the maximum amount that must be spent to use the coupon.
    • Individual Use: Specifies whether the coupon can be used with different coupons.
    • Exclude Sale Items: Specifies whether the coupon must be applied to items on sale.

    Usage Limit FastTab

    The following list includes information about the fields on the Usage Limit FastTab:

    • Usage Limit: Specifies the number of times the coupon can be used.
    • Limit Usage to X Items: Specifies the maximum number of individual items to which the coupon can apply when using product discounts.
    • Limit Usage per User: Specifies the number of times an individual user can use the coupon.
